== Description ==

The **Personal Dictionary plugin** helps you to create and organize your own vocabulary lists, study and memorize the words relating to that specific subject. The plugin is a must-have tool for school teachers, university professors, language tutors, or self-educational purposes. It can be considered an integral part of any learning management. Take your learning process to an advanced level. With the help of us, you can make your study activity more organized and efficient.

How does the plugin work? You, as a WordPress admin, just need to copy and paste the shortcode of the plugin into any post/page and the tool will be activated.

> Dictionary shortcode <br>
> [ayspd_userpage]

After activating the tool on the front-end, each user will see individually presented content based on their created groups and words with their corresponding translations within those groups. Then, comes the fun part: Games!
There are two types of games available: Find the Word and Find the Translation.

== Installation ==


There are two ways to install Personal Dictionary: the easy way, when you install Personal Dictionary from your WordPress dashboard, and the not so easy way, when you install it from WordPress.org.

* 1.1 The easiest way to enjoy Personal Dictionary:
* 1.1.1	Login to your WordPress dashboard
* 1.1.2	Go to Plugins
* 1.1.3	Add New
* 1.1.4	Search for Personal Dictionary
* 1.1.5	Click to install
* 1.2 The second way:
* 1.2.1Download the zip file from https://wordpress.org/plugins/personal-dictionary/
* 1.2.2 Go to Plugins
* 1.2.3 Add New
* 1.2.4 Upload plugin
* 1.2.5 Choose file ays personal_dictionary.zip
* 1.2.6	Click to install

* 1.3 In order to install the Personal Dictionary from Wordpress.org you must unzip the archive and copy the personal-dictionary folder into your plugins folder (\wp-content\plugins).
After one ofthis steps you can activate the plugin from the Plugins menu.
Once activated configure any options as desired and you can enjoy our Personal Dictionary.
